# Privacy Policy

**agentic-study-environment** is a Claude Code plugin. It consists entirely of
markdown skill definitions that run locally inside Claude Code. It does not:

- collect, transmit, or store any user data
- make network requests to any external service
- send source materials, session transcripts, or progress logs anywhere

All data (source materials, curricula, progress logs, work files) stays on your
local machine in the `.studyenv/` directory the plugin creates. The plugin has
no server-side component.

Questions: open an issue at https://github.com/TimboGP/agentic-study-environment/issues
